  <Steps>
    <Step title="Install Craft Agents">
      Download and install using our one-line installer:
      
      ```bash macOS/Linux
      curl -fsSL https://agents.craft.do/install-app.sh | bash
      ```
      
      ```powershell Windows
      irm https://agents.craft.do/install-app.ps1 | iex
      ```
    </Step>
    
    <Step title="Choose your AI provider">
      Connect to your preferred LLM provider:
      - Anthropic (API key or Claude Max)
      - Google AI Studio
      - ChatGPT Plus (Codex OAuth)
      - GitHub Copilot OAuth
      
      You can add multiple providers and switch between them per workspace.
    </Step>
    
    <Step title="Create a workspace">
      Set up your first workspace to organize sessions and configure sources. Workspaces let you:
      - Organize sessions by project or context
      - Configure MCP servers and API sources
      - Set custom skills and permissions
      - Define automation rules
    </Step>
    
    <Step title="Start your first session">
      Create a new chat session and interact with your AI agent. Sessions support:
      - File attachments (drag-drop images, PDFs, Office docs)
      - Real-time tool visualization
      - Background task execution
      - Multi-file diff viewer
    </Step>
  </Steps>
</div>
